Privacy Policy

Last updated: June 20, 2026

Thank you for adding Gh0st Bot to your server. Your privacy is dynamic and very important to us. This Privacy Policy outlines what data we collect, how it is used, and how we protect it.

1. Data We Collect

By using the bot, certain information provided by the Discord API may be processed temporarily or stored to ensure proper functionality:

  • User IDs and Usernames: To track application systems, ticket ownership, logs, or custom moderation features (like bans).
  • Server/Guild IDs: To save specific configuration settings, permission structures, and custom commands.
  • Message Content: Only processed reactively when invoking a command (e.g., Slash Commands or specified form submittals). We do not log or store your standard chat history.

2. How We Use Data

The data collected is used strictly to provide and improve the bot’s core operations:

  • Executing moderation commands properly (such as target user lookup for ban commands).
  • Managing access controls and visibility permissions for private ticket channels.
  • Logging moderation actions inside designated channels for server administrators.

3. Data Retention and Deletion

We do not keep data longer than necessary. Settings and logs are preserved only as long as the bot remains on your server. If you wish to have all stored data related to your server or user profile manually wiped, please contact the Bot Owner directly via Discord.

4. Third-Party Services

Our bot's platform logic is built using BotGhost. Consequently, processing complies entirely with Discord’s Developer Terms of Service and BotGhost’s underlying infrastructure safeguards.

5. Contact Us

If you have any questions, concerns, or requests regarding this Privacy Policy, please reach out to the development staff inside our support server.
